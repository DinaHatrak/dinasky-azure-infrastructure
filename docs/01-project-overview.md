# Azure Infrastructure for DinaSky Technologies

## Project Overview

This project demonstrates the design, deployment, and management of a secure Microsoft Azure infrastructure for a fictional aviation technology company.

The objective is to simulate a real-world enterprise Azure environment by implementing networking, virtual machines, storage, monitoring, identity management, and security best practices.

## Project Objectives

- Build a secure Azure infrastructure.
- Design scalable virtual networks.
- Deploy and manage Azure Virtual Machines.
- Configure Azure Storage.
- Implement monitoring.
- Apply Role-Based Access Control (RBAC).
- Document the entire deployment process.
